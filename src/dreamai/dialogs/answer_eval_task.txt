Given:
1. The user's original query
2. The source documents used to generate the answer (assume these are correct and factual)
3. The generated answer to that query

Your task is to analyze the query, the source documents, and the answer, then provide a boolean response indicating whether the answer adequately addresses the user's query and is based on the source documents, along with a short explanation of your reasoning.

Instructions:
1. Carefully examine the user's query to understand the main question, intent, and any specific requirements.
2. Review the source documents to understand what information was available to generate the answer.
3. Analyze the generated answer in relation to both the query and the source documents.
4. Determine if the answer directly addresses the core aspects of the user's query AND is grounded in the information provided by the source documents.
5. Ensure that the answer does not contain significant information that is not present in the source documents.
6. Provide a short, clear explanation of your reasoning for the evaluation.
7. Do NOT question the correctness of the source documents, even if you believe they may be outdated or inaccurate. Treat all information in the source documents as factual and up-to-date.

Evaluation Criteria:
- Return True if:
  a. The answer directly addresses the main point(s) of the user's query.
  b. The information in the answer is clearly derived from the source documents.
  c. The answer fulfills any specific requirements mentioned in the query (e.g., if the query asks for a list, the answer provides a list based on the source documents).

- Return False if:
  a. The answer is off-topic or addresses a different question than what was asked.
  b. The answer contains significant information that is not present in the source documents.
  c. The answer is too vague or general to be considered a real attempt at addressing the specific query.
  d. The answer ignores key aspects or requirements of the query, despite relevant information being available in the source documents.

Output:
Provide a JSON object with the following structure:
{
  "evaluation": boolean,
  "reasoning": "A short sentence explaining your evaluation"
}

The boolean value should be:
- true if the answer addresses the query and is grounded in the source documents
- false if the answer does not address the query or is not based on the source documents

Your task is to make a judgment based on both query-answer relevance and the answer's grounding in the source documents. Ensure your reasoning reflects this dual focus. Remember, do not question or comment on the accuracy or currency of the source documents - treat them as absolute truth for this evaluation.
